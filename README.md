# GU-BIOINFORMATICS-MS
Mark And Sameel's Genomics Project

The goal of this project was to evaluate the viral diversity in a sample from Palsa permafrost in northern Sweden. We acquired raw reads from another paper titled "Soil Viruses Are Underexplored Players in Ecosystem Carbon Processing" (Trubl et al. 2018). We then trimmed these raw reads, assembled viral contigs, identified vOTUS, and analyzed the diversity in our sample using R.
